# taylor-approximation-NumPy
Python project in numerical methods and polynomial approximation using NumPy and Matplotlib. Includes work with sequences, Taylor coefficients, polynomial evaluation, polynomial multiplication, and graphical comparison between exact functions and approximations such as  𝑒 ^𝑥 and  sin(𝑥).





# Se: Taylor-Approximation-NumPy

Utvecklade ett Python-program med NumPy och Matplotlib för att arbeta med talföljder, Taylor-koefficienter, polynomutvärdering och approximation av elementära funktioner. Implementerade även polynommultiplikation, jämförde approximationer med exakta funktionsvärden och visualiserade resultaten grafiskt.

# English:
# Taylor Approximation in Python

A Python project using NumPy and Matplotlib to work with sequences, Taylor coefficients, polynomial evaluation, and approximation of elementary functions. Also includes polynomial multiplication, comparison with exact function values, and graphical visualization of the results.

Detailed:
Innehåll:
- Arbete med NumPy-arrayer och kumulativa summor
- Beräkning av koefficienter för exponential- och sinusserier
- Konstruktion av Fibonacci-tal i arrayform
- Utvärdering av polynom i givna punkter
- Approximation av \(e^2\) och \(\sin(5\pi/3)\) med Taylorpolynom
- Grafisk jämförelse mellan exakta funktioner och approximationer
- Implementering av polynommultiplikation och kontroll mot NumPy:s polynomverktyg

Programmet är skrivet i Python och använder NumPy samt Matplotlib.
